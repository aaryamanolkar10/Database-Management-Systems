DBMS Lab Programs

This repository contains Database Management System (DBMS) programs implemented using Oracle SQL and SQL*Plus. The programs demonstrate database design, table creation, constraints, queries, and various SQL operations commonly performed in DBMS practicals and mini projects.

Features:

Database creation using SQL
Table creation with constraints
Primary Key and Foreign Key implementation
CHECK and NOT NULL constraints
INSERT, UPDATE, DELETE operations
ALTER TABLE operations
SQL queries and outputs
SQL*Plus compatible programs

Technologies Used:

Oracle Database 21c Express Edition
SQL*Plus
SQL

Concepts Covered:

Relational Database Design
Constraints in DBMS
DDL and DML commands
Referential Integrity
Table Modification
Database Operations

Purpose:
This repository is created for DBMS practicals, SQL practice, academic reference, and learning Oracle SQL concepts.
